# Basic of java
Java is a high-level, class-based, object-oriented programming language designed to have as few implementation dependencies as possible. 
It operates on the core principle of "Write Once, Run Anywhere" (WORA),
meaning compiled Java code can run on all platforms that support Java without the need for recompilation.

Core Environment ComponentsTo develop and run programs,
Java relies on three core sub-systems:
JVM (Java Virtual Machine): Executes the compiled bytecode on your hardware.
JRE (Java Runtime Environment): Combines the JVM with core libraries to run apps.
JDK (Java Development Kit): Includes the JRE and compiler (javac) to build apps

Features of Java
Simple – Easy to learn and use.
Object-Oriented – Based on classes and objects.
Platform Independent – "Write Once, Run Anywhere" (WORA).
Secure – Provides security features like bytecode verification.
Robust – Strong memory management and exception handling.
Multithreaded – Supports concurrent execution of tasks.
